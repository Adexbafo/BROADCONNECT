# BROADCONNECT Protocol

**Identity Layer | Social Graph | Cross-Chain Application Platform**

BROADCONNECT is a decentralized social protocol built on the **Supra** and **Base** networks. It introduces a unified cross-chain identity standard (**BCID-1**) and a programmable social graph organized into **ROOMS**.

## 🌐 Vision
To bridge the high-performance vertical stack of Supra with the liquid ecosystem of Base, providing a seamless, mobile-first social experience where users own their identity and data across chains.

## 🛠 Tech Stack
- **Framework:** Laravel 11 (PHP 8.3+)
- **Chains:** Supra (L1) & Base (L2)
- **Database:** PostgreSQL (with Redis for BroaderAgent indexing)
- **Frontend:** Tailwind CSS & Livewire (Mobile-First SPA)
- **Protocol Intelligence:** BroaderAgent (Off-chain indexing & Reputation)

## 🏗 Architecture (BCID-1)
- **Identity:** One BCID-1 links one Supra address and one EVM (Base) address.
- **Registration:** $5 USD fee (Payable in SUPRA, USDC, or ETH).
- **Treasury:** Fees are funneled to reward active users and provide liquidity for the BC-SOC token.
- **Social Graph:** Interaction happens within **ROOMS**.

## 🚀 Roadmap
- **Phase 1: Identity Core (Current)** - BCID-1 Registry & Multi-Chain Wallet Linking.
- **Phase 2:** ROOMS Module & Staking logic.
